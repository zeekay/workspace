## Develop in comfort

[![Greenkeeper badge](https://badges.greenkeeper.io/zeekay/workspace.svg)](https://greenkeeper.io/)
Workspace makes it possible to work on multiple, interdependent JavaScript
libraries at once.

### Install
```bash
$ npm install workspace -g
```
